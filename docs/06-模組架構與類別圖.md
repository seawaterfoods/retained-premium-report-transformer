# 第六章：模組架構與類別圖

---

## 1. 類別圖

```mermaid
classDiagram
    class ReportGeneratorRunner {
        -ReportGeneratorService reportGenerator
        -AppConfig appConfig
        +run(String... args)
    }

    class ReportGeneratorService {
        -FilenameParserService filenameParser
        -ExcelReaderService excelReader
        -DataTransformerService dataTransformer
        -TemplateWriterService templateWriter
        -LastYearReaderService lastYearReader
        -ValidationService validator
        +generate(inputDir, templatePath, outputDir, lastYearDir)
        -scanInputFiles(inputDir) List~File~
    }

    class FilenameParserService {
        -Pattern FILENAME_PATTERN
        +parse(filepath) Optional~FileInfo~
    }

    class ExcelReaderService {
        +readSourceFile(filePath) Optional~CompanyData~
        ~getCellStringValue(cell) String
        -evaluateNumericCell(cell, evaluator) double
    }

    class DataTransformerService {
        +groupByQuarter(fileInfos, companyDataMap) Map~Integer_QuarterData~
        +aggregateCategories(companyData) Map~Integer_Double~
    }

    class TemplateWriterService {
        -DataTransformerService dataTransformerService
        +writeReport(templatePath, outputPath, quarterDataMap, lastYearData, year, maxQuarter)
        -buildCompanyRowMap(sheet, dataStart, dataEnd) Map~String_Integer~
        -setCellNumericValue(cell, value)
    }

    class LastYearReaderService {
        +readLastYearData(currentYear, quarter, lastYearDir) Map~String_Double~
        -getCompanyCode(cell) String
        -evaluateNumericCell(cell, evaluator) double
    }

    class ValidationService {
        +validateFileInfos(fileInfos) ValidationResult
        +validateCompanyData(data, filename) ValidationResult
        +validateAll(fileInfos, companyDataMap) ValidationResult
    }

    class AppConfig {
        -String inputDir
        -String outputDir
        -String templatePath
        -String lastYearDir
    }

    class FileInfo {
        <<record>>
        +String companyCode
        +int year
        +int startMonth
        +int endMonth
        +int quarter
        +String filename
    }

    class CompanyData {
        <<record>>
        +String companyCode
        +String companyName
        +Map~String_Double~ retainedPremiums
    }

    class QuarterData {
        <<record>>
        +int year
        +int quarter
        +int startMonth
        +int endMonth
        +Map~String_CompanyData~ companies
    }

    class ValidationResult {
        -boolean passed
        -List~String~ errors
        -List~String~ warnings
        +addError(msg)
        +addWarning(msg)
        +merge(other)
    }

    ReportGeneratorRunner --> ReportGeneratorService
    ReportGeneratorRunner --> AppConfig
    ReportGeneratorService --> FilenameParserService
    ReportGeneratorService --> ExcelReaderService
    ReportGeneratorService --> DataTransformerService
    ReportGeneratorService --> TemplateWriterService
    ReportGeneratorService --> LastYearReaderService
    ReportGeneratorService --> ValidationService
    TemplateWriterService --> DataTransformerService
    FilenameParserService ..> FileInfo
    ExcelReaderService ..> CompanyData
    DataTransformerService ..> QuarterData
    ValidationService ..> ValidationResult
```
