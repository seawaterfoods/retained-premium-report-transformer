---

## 1. 錯誤分類表

```mermaid
graph TD
    subgraph FATAL["🔴 中止流程"]
        E2["同公司同季度重複"]
        E3["年度不一致"]
        E4["來源資料讀取失敗"]
        E5["報表寫入失敗"]
    end

    subgraph WARN["🟡 警告繼續"]
        W1["去年檔案不存在<br/>(U 欄留空)"]
        W2["負值保費<br/>(可能為調整)"]
        W3["險種數 ≠ 33"]
    end

    subgraph SKIP["🟠 跳過此檔"]
        S1["檔名格式不符"]
    end

    style FATAL fill:#ffebee,stroke:#f44336
    style WARN fill:#fff8e1,stroke:#ffc107
    style SKIP fill:#fff3e0,stroke:#ff9800
```

## 2. 日誌等級對照

| 情境 | Log 等級 | 行為 |
|------|----------|------|
| 檔名格式不符 | `ERROR` | 跳過此檔，繼續 |
| 來源 Excel 結構異常 | `ERROR` | 跳過此檔，中止流程 |
| 來源資料非數值 | `ERROR` | 跳過此檔，中止流程 |
| 同公司同季度重複 | `ERROR` | 中止整個流程 |
| 年度不一致 | `ERROR` | 中止整個流程 |
| 寫入/輸出失敗 | `ERROR` | 中止整個流程 |
| 去年檔案不存在 | `WARN` | U 欄留空，繼續 |
| 驗算差異 (F≠C+D-E) | `WARN` | 記錄但繼續 |
| 正常處理步驟 | `INFO` | 記錄進度 |
