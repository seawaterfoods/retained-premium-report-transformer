## 2. 報表產生邏輯

報表透過 Apache POI (XSSFWorkbook) 程式化產生，由 ReportWriter 搭配 ExcelStyleHelper 建立完整的 Excel 報表。

### 2.1 Sheet1 區塊佈局

```mermaid
graph TD
    subgraph Sheet1["Sheet1: {年度}自留(季)"]
        H["Row 1-2: 標題列"]
        Q1["Q1 區塊<br/>Row 3~21 (公司資料)<br/>Row 22 (SUBTOTAL 小計)"]
        Q2["Q2 區塊<br/>Row 23~41 (公司資料)<br/>Row 42 (SUBTOTAL 小計)"]
        Q3["Q3 區塊<br/>Row 43~61 (公司資料)<br/>Row 62 (SUBTOTAL 小計)"]
        Q4["Q4 區塊<br/>Row 63~81 (公司資料)<br/>Row 82 (SUBTOTAL 小計)"]
    end

    H --> Q1 --> Q2 --> Q3 --> Q4
```

### 2.2 Sheet2 區塊佈局

```mermaid
graph TD
    subgraph Sheet2["Sheet2: {年度}自留總累"]
        H2["Row 1: 動態標題<br/>Row 2-5: 欄位標題"]
        Q1S2["Q1 區塊<br/>Row 6~24 (公司資料)<br/>Row 25 (SUBTOTAL 小計)"]
        Q2S2["Q2 區塊<br/>Row 26~44 (公司資料)<br/>Row 45 (SUBTOTAL 小計)"]
        Q3S2["Q3 區塊<br/>Row 46~64 (公司資料)<br/>Row 65 (SUBTOTAL 小計)"]
        Q4S2["Q4 區塊<br/>Row 66~84 (公司資料)<br/>Row 85 (SUBTOTAL 小計)"]
        GT["Row 86: 總計 (SUBTOTAL)"]
        CT["Row 87: 筆數"]
    end

    H2 --> Q1S2 --> Q2S2 --> Q3S2 --> Q4S2 --> GT --> CT
```

### 2.3 公式保留策略

```mermaid
graph LR
    subgraph Write["✏️ 程式產生"]
        W1["Sheet1 D~AJ 欄<br/>(33 險種值)"]
        W3["Sheet2 U 欄<br/>(去年同期)"]
    end

    subgraph Keep["🔒 程式建立的公式"]
        K1["Sheet1 AK 欄<br/>=SUM(D:AJ)"]
        K2["所有小計列<br/>=SUBTOTAL(9,...)"]
        K6["Sheet2 D~S 欄<br/>跨Sheet公式(引用Sheet1)"]
        K3["Sheet2 T 欄<br/>=SUM(D:S)"]
        K4["Sheet2 V 欄<br/>=IF(U=0,'',(T/U)-1)"]
        K5["Sheet2 Row86<br/>=SUBTOTAL(9,全部)"]
    end

    style Write fill:#e8f5e9,stroke:#4CAF50
    style Keep fill:#fff3e0,stroke:#FF9800
```

### 2.4 動態公司隱藏機制

```mermaid
stateDiagram-v2
    [*] --> 建立報表: ReportWriter 程式化產生

    建立報表 --> 寫入來源公司: 從來源資料取得公司清單

    state 寫入來源公司 {
        有資料 --> 寫入數值: 填入自留保費
        無資料 --> 隱藏列: setZeroHeight(true)
    }

    寫入數值 --> SUBTOTAL自動計算
    隱藏列 --> SUBTOTAL自動計算

    note right of 隱藏列
        SUBTOTAL(9,...) 會自動
        忽略隱藏列，確保
        小計/總計正確
    end note

    SUBTOTAL自動計算 --> [*]: 輸出報表
```
