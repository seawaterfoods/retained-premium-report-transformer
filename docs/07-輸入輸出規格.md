# 第七章：輸入輸出規格

---

## 1. 來源檔名格式

```
{公司別}_{年度}({起始月}-{結束月})_自留保費統計表.xlsx
```

```mermaid
graph LR
    FN["29_115(01-03)_自留保費統計表.xlsx"]
    FN --> C["公司別: 29"]
    FN --> Y["年度: 115"]
    FN --> SM["起始月: 01"]
    FN --> EM["結束月: 03"]
    FN --> Q["季度: Q1"]
```

### 季度判定規則（累計制）

| 起始月 | 結束月 | 季度 | 月份標記 |
|--------|--------|------|----------|
| 01 | 03 | Q1 | `1-1Q(1-3)` |
| 01 | 06 | Q2 | `1-2Q(1-6)` |
| 01 | 09 | Q3 | `1-3Q(1-9)` |
| 01 | 12 | Q4 | `1-4Q(1-12)` |

## 2. 來源檔 Excel 結構

```mermaid
graph TD
    subgraph Source["來源 Excel 結構"]
        R1["Row 1: 標題"]
        R4["Row 4: B4=公司代號, C4=公司名稱"]
        R5["Row 5: 欄位標題"]
        R7["Row 7~39: 33 種險種資料"]
        R40["Row 40: 合計列"]
    end

    subgraph Cols["欄位"]
        A["A: 險種代號"]
        B["B: 險種名稱"]
        C["C: 簽單保費收入"]
        D["D: 分進再保收入"]
        E["E: 分出再保支出"]
        F["F: 自留保費 ✨<br/>(程式讀取此欄)"]
        G["G: 自留比率"]
    end
```

## 3. 輸出檔名格式

```
{年度}年產險業務(Q{最大季度}季自留)保費統計表.xlsx
```

## 4. 輸出 Excel 結構

| Sheet | 名稱 | 用途 |
|-------|------|------|
| Sheet 1 | `{年度}自留(季)` | 各公司 × 33 險種明細 |
| Sheet 2 | `{年度}自留總累` | 各公司 × 16 分類彙總 + 去年對比 |
| Sheet 3 | `歸屬` | 險種分類對照表（不修改） |

---

## 5. 資料流圖

### 5.1 資料轉換全流程

```mermaid
flowchart LR
    subgraph Input
        F1["來源檔 A<br/>(公司 29)"]
        F2["來源檔 B<br/>(公司 05)"]
        F3["來源檔 C<br/>(公司 01)"]
    end

    subgraph Parse["解析"]
        FI1["FileInfo<br/>code=29, Q1"]
        FI2["FileInfo<br/>code=05, Q1"]
        FI3["FileInfo<br/>code=01, Q1"]
    end

    subgraph Read["讀取"]
        CD1["CompanyData<br/>33 險種保費"]
        CD2["CompanyData<br/>33 險種保費"]
        CD3["CompanyData<br/>33 險種保費"]
    end

    subgraph Transform["轉換"]
        QD["QuarterData<br/>Q1: 3 家公司"]
        AGG["聚合分類<br/>33 → 16 類"]
    end

    subgraph Write["寫入"]
        S1["Sheet1<br/>33 險種明細"]
        S2["Sheet2<br/>16 分類彙總"]
    end

    F1 --> FI1 --> CD1
    F2 --> FI2 --> CD2
    F3 --> FI3 --> CD3
    CD1 --> QD
    CD2 --> QD
    CD3 --> QD
    QD --> S1
    QD --> AGG --> S2
```

### 5.2 Sheet1 寫入映射

```mermaid
graph TB
    subgraph Source["來源檔 (公司 29)"]
        SF["F 欄 (自留保費)"]
        SC["代號: 0100=100萬, 0200=50萬, ..."]
    end

    subgraph Template["模板 Sheet1"]
        direction TB
        R1["Row 1: 險種代號 0100  0200  ...  9900"]
        R2["Row 2: 欄標題    D     E    ...  AJ    AK"]
        R3["Row 3: 公司 01"]
        R18["Row 18: 公司 29 ← 寫入此列"]
        R22["Row 22: 小計 (SUBTOTAL 公式)"]
    end

    SF -->|"0100 → D18"| R18
    SF -->|"0200 → E18"| R18
    SF -->|"9900 → AJ18"| R18
```
