# 第二章：安裝與部署

---

## 1. 本機執行 (開發/除錯)

```bash
# 編譯
build.bat
# 或
mvnw.cmd clean package -DskipTests

# 執行（使用 config/application.yml 設定）
run.bat
# 或
java -jar target\retained-premium-report-transformer-0.0.1-SNAPSHOT.jar --spring.config.additional-location=file:./config/
```

## 2. Docker 部署

```bash
# 建置映像檔
docker-compose build

# 執行
docker-compose up

# 背景執行
docker-compose up -d
```

### Docker 目錄掛載

```
./import/      → /data/import      (來源檔案)
./output/      → /data/output      (輸出目錄)
./config/      → /app/config       (設定檔)
```
