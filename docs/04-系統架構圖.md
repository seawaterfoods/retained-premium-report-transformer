# 第四章：系統架構圖

---

## 1. 整體架構

```mermaid
graph TB
    subgraph Input["📂 輸入"]
        SRC1["29_115(01-03)_自留保費統計表.xlsx"]
        SRC2["05_115(01-03)_自留保費統計表.xlsx"]
        SRC3["01_115(01-03)_自留保費統計表.xlsx"]
        TMPL["模板.xlsx"]
        LAST["去年報表.xlsx"]
    end

    subgraph App["⚙️ 報表轉換系統 (Spring Boot)"]
        GEN["ReportGenerationService<br/>流程串接 + CLI 進入點"]
        FP["FilenameParser<br/>檔名解析"]
        ER["ExcelSourceReader<br/>Excel 讀取"]
        DT["DataTransformerService<br/>資料轉換"]
        TW["TemplateWriter<br/>模板寫入"]
        LY["LastYearReader<br/>去年資料"]
        FV["FileValidator<br/>驗證"]
    end

    subgraph Output["📄 輸出"]
        OUT["115年產險業務(Q1季自留)保費統計表.xlsx"]
        LOG["應用程式日誌"]
    end

    SRC1 --> FP
    SRC2 --> FP
    SRC3 --> FP
    FP --> GEN
    SRC1 --> ER
    SRC2 --> ER
    SRC3 --> ER
    ER --> GEN
    GEN --> VS
    GEN --> DT
    LAST --> LY
    LY --> GEN
    GEN --> TW
    TMPL --> TW
    TW --> OUT
    GEN --> LOG
```
