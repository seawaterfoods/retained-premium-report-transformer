# 自留保費統計表報表轉換系統 — 規格文件

## 專案目標

將多個產險公司的「自留保費統計表」Excel 來源檔，自動彙整填入輸出模板，產生季度/累計報表。

- **輸入**：多個 `公司別_年度(月-月)_自留保費統計表.xlsx`
- **輸出**：`{年度}年產險業務(Q{季}季自留)保費統計表.xls`

## 3. 檔名解析規則

### 來源檔名格式
```
公司別_年度(起始月-結束月)_自留保費統計表.xlsx
```

### Regex
```
^(\d{1,2})_(\d{2,3})\((\d{2})-(\d{2})\)_自留保費統計表\.xlsx$
```

### 欄位拆解

| 群組 | 名稱 | 範例值 | 說明 |
|------|------|--------|------|
| `\1` | 公司別 | `29` | 公司代號（1-2 位數字） |
| `\2` | 年度 | `115` | 民國年（2-3 位數字） |
| `\3` | 起始月 | `01` | 固定 2 位數 |
| `\4` | 結束月 | `03` | 固定 2 位數 |

### 錯誤格式處理策略

| 錯誤類型 | 處理方式 |
|----------|----------|
| 檔名不符 Regex | **中止**，記錄錯誤訊息，跳過此檔 |
| 公司代號不在已知清單 | **警告**，仍嘗試處理 |
| 月份不合理（>12 或 起始>結束） | **中止**此檔 |
| 副檔名非 `.xlsx` | **跳過**，僅處理 `.xlsx` |

## 4. 季度判斷與輸出命名規則

### 月份區間 → 季度

| 起始月 | 結束月 | 季度 | 月份標記（Sheet1） | 月份標記（Sheet2） |
|--------|--------|------|--------------------|--------------------|
| 01 | 03 | Q1 | `1-1Q(1-3)` | `1-1Q(1-3)` |
| 01 | 06 | Q2 | `1-2Q(1-6)` | `1-2Q(1-6)` |
| 01 | 09 | Q3 | `1-3Q(1-9)` | `1-3Q(1-9)` |
| 01 | 12 | Q4 | `1-4Q(1-12)` | `1-4Q(1-12)` |

> **注意**：月份標記的格式為**累計制**，非單季制。Q2 = 1~6 月累計，非 4~6 月。

### 季度判斷邏輯

```
if 起始月 == '01':
    if 結束月 == '03': quarter = 1
    elif 結束月 == '06': quarter = 2
    elif 結束月 == '09': quarter = 3
    elif 結束月 == '12': quarter = 4
    else: ERROR
else:
    ERROR（起始月非01，不符預期格式）
```

## 7. 公式處理策略

### 來源檔公式分析

| 欄位 | 公式 | 處理策略 |
|------|------|----------|
| F (自留保費) | `=SUM(C+D-E)` | **讀取計算結果值**（`data_only=True`），不需保留公式 |
| G (自留比率) | `=IF((C+D)=0,0,F/(C+D))` | **不使用**，輸出模板無此欄 |
| C40/D40/E40 (合計) | `=SUM(C7:C39)` | **不使用**，僅讀取個別險種值 |

### 輸出模板公式分析

由於 `.xls` 格式限制，xlrd 無法直接讀取公式文字。根據資料特徵推測：

| 區域 | 是否有公式 | 推測依據 | 處理策略 |
|------|-----------|----------|----------|
| Sheet1 AK 欄 (合計) | **可能有** SUM(D:AJ) | Q2-Q4 值為 0 | **TODO** |
| Sheet1 小計列 (Row 22/42/62/82) | **可能有** SUM | Q1 有值,Q2-Q4 為 0 | **TODO** |
| Sheet2 T 欄 (年度合計) | **可能有** SUM(D:S) | 值=各欄加總 | **TODO** |
| Sheet2 V 欄 (成長率) | **可能有** (T-U)/U | Q2-Q4 值=-1 | **TODO** |
| Sheet2 小計列 | **可能有** SUM | 同上 | **TODO** |
| Sheet2 總計列 | **可能有** SUM | 同上 | **TODO** |

### 推薦策略

**採用「全部由程式寫值」模式**：

理由：
1. `.xls` 格式中的公式無法被 xlrd 讀取確認
2. 使用 `xlutils` 或 `xlwt` 寫入 `.xls` 時，保留原公式的技術複雜度高
3. 程式計算寫值可確保正確性且可驗證

具體做法：
- AK 欄：程式計算 SUM(D:AJ) 後寫入
- 小計列：程式計算各公司加總後寫入
- T 欄：程式計算 SUM(D:S) 後寫入
- V 欄：程式計算 (T-U)/U 後寫入（注意 U=0 時處理）
- 總計列：程式計算各季小計加總後寫入

### 風險

| 風險 | 影響 | 緩解 |
|------|------|------|
| 覆蓋原有公式 | 後續人工修改資料時無法自動重算 | 可在完成後用 openpyxl 轉為 .xlsx 並重建公式 |
| 計算精度差異 | Python float vs Excel float | 使用 Decimal 或四捨五入到整數 |

### 任務清單

#### Task 1：檔名解析模組 (`filename_parser.py`)
- **輸入**：檔案路徑或檔名字串
- **輸出**：`{ company_code, year, start_month, end_month, quarter }`
- **功能**：
  - Regex 解析檔名
  - 季度判斷
  - 錯誤處理（格式不符返回 None + 錯誤訊息）
- **依賴**：無

#### Task 2：常數與模型定義 (`constants.py`, `models.py`)
- **內容**：
  - 19 家公司清單 (代號→名稱)
  - 33 種險種代號清單
  - 險種代號→模板欄位位置 mapping
  - 險種分類歸屬規則 (Sheet2 用)
  - 模板列位 mapping (公司代號→各季度列號)
  - 資料模型 dataclass 定義
- **依賴**：無

#### Task 3：Excel 讀取模組 (`excel_reader.py`)
- **輸入**：來源 `.xlsx` 檔案路徑
- **輸出**：結構化資料 `{ company_code, company_name, year, month_range, insurance_data: { code: retained_premium } }`
- **功能**：
  - 使用 `openpyxl` 讀取（`data_only=True`）
  - 提取公司代號 (B4)、公司名稱 (C4)
  - 提取各險種自留保費 (F7:F39)，以險種代號為 key
  - 資料型別轉換與驗證
- **依賴**：Task 2

#### Task 4：資料轉換模組 (`data_transformer.py`)
- **輸入**：多個公司的結構化資料
- **輸出**：
  - Sheet1 資料矩陣（公司×險種）
  - Sheet2 資料矩陣（公司×分類彙總）
  - 小計/總計列資料
- **功能**：
  - 險種歸類彙總（33 種 → 16 分類）
  - 合計欄計算 (AK / T)
  - 小計列計算
  - 總計列計算
  - 成長率計算
- **依賴**：Task 2, Task 3

#### Task 5：模板寫入模組 (`template_writer.py`)
- **輸入**：轉換後的資料矩陣 + 模板檔案路徑
- **輸出**：寫入完成的模板檔案（記憶體中的 workbook 物件）
- **功能**：
  - 使用 `xlrd` 讀取模板 + `xlutils.copy` 建立可寫入副本
  - 定位目標儲存格（公司代號→列號、險種代號→欄號）
  - 寫入 Sheet1 明細資料
  - 寫入 Sheet2 彙總資料
  - 保留原有格式
  - 更新標題（年度/季度）
- **依賴**：Task 2, Task 4
- **技術注意**：`xlutils.copy` 可保留格式，但可能遺失部分進階格式

#### Task 6：驗證模組 (`validator.py`)
- **輸入**：寫入完成的 workbook
- **輸出**：驗證報告 `{ passed: bool, errors: [], warnings: [] }`
- **功能**：
  - 逐列驗算合計
  - 逐欄驗算小計
  - 成長率驗算
  - 空值/異常值檢查
- **依賴**：Task 2, Task 5

#### Task 7：輸出模組 (`output_generator.py`)
- **輸入**：驗證通過的 workbook + 輸出路徑
- **輸出**：`.xls` 檔案
- **功能**：
  - 生成輸出檔名
  - 儲存檔案
  - 產生處理報告（含驗證結果、處理的公司清單、跳過的檔案）
- **依賴**：Task 5, Task 6

#### Task 8：主程式 (`main.py`)
- **功能**：
  - 接受輸入目錄/檔案清單 + 模板路徑 + 輸出目錄
  - 掃描來源檔案
  - 呼叫各模組
  - 錯誤匯總與輸出
  - CLI 介面（argparse）
- **依賴**：所有模組

### 技術選型

| 需求 | 工具 | 說明 |
|------|------|------|
| 讀取 `.xlsx` 來源檔 | `openpyxl` (data_only=True) | 標準選擇 |
| 讀取 `.xls` 模板 | `xlrd` (formatting_info=True) | 需支持格式資訊 |
| 寫入 `.xls` 輸出 | `xlutils.copy` + `xlwt` | 保留模板格式 |
| CLI 介面 | `argparse` | Python 內建 |
| 日誌 | `logging` | Python 內建 |
| 測試 | `pytest` | 標準選擇 |

> **替代方案**：若可以輸出 `.xlsx`，則可全部使用 `openpyxl`，技術複雜度大幅降低。
