# 第三章：設定與使用方式

---

## 1. 設定說明

### 1.1 application.yml

外部設定檔位於 `config/application.yml`：

```yaml
app:
  import-dir: ./import                         # 來源檔案目錄
  output-dir: ./output                         # 輸出目錄
  template-path: ./import/template.xlsx        # 模板路徑
  last-year-dir: ./import/lastyear             # 去年報表目錄
```

### 1.2 覆蓋優先順序

```
外部 config/application.yml > 環境變數 > 內建 application.yml
```

| 參數位置 | application.yml | 環境變數 |
|----------|-----------------|----------|
| 匯入目錄 | `app.import-dir` | `APP_IMPORT_DIR` |
| 模板路徑 | `app.template-path` | `APP_TEMPLATE_PATH` |
| 輸出目錄 | `app.output-dir` | `APP_OUTPUT_DIR` |
| 去年目錄 | `app.last-year-dir` | `APP_LAST_YEAR_DIR` |

---

## 2. 使用方式

### 2.1 基本用法

1. 將所有來源 `.xlsx` 檔案放入 `import/` 目錄
2. 確認 `import/template.xlsx` 存在
3. 執行程式
4. 從 `output/` 取得產出報表

### 2.2 多公司匯入

放入多個來源檔案，系統會自動合併：

```
import/
├── 01_115(01-03)_自留保費統計表.xlsx    # 台產
├── 05_115(01-03)_自留保費統計表.xlsx    # 富邦
├── 29_115(01-03)_自留保費統計表.xlsx    # 美國國際
└── 32_115(01-03)_自留保費統計表.xlsx    # 美商安達
```

輸出：`115年產險業務(Q1季自留)保費統計表.xlsx`（僅顯示這 4 家公司，其餘隱藏）

### 2.3 多季度匯入

同時放入不同季度的檔案，系統會自動填入對應區塊：

```
import/
├── 29_115(01-03)_自留保費統計表.xlsx    # Q1
├── 29_115(01-06)_自留保費統計表.xlsx    # Q2
└── 05_115(01-03)_自留保費統計表.xlsx    # Q1
```

輸出：`115年產險業務(Q2季自留)保費統計表.xlsx`（檔名用最大季度）

### 2.4 去年同期對比

將去年報表放入 `import/lastyear/` 目錄：

```
import/
└── lastyear/
    └── 114年產險業務(Q1季自留)保費統計表.xlsx
```

系統會自動讀取 Sheet2 的 T 欄值填入 U 欄（去年同期合計），並計算成長率。

> ⚠️ 若去年檔案不存在，U 欄留空，日誌記錄 WARN，不影響其他功能。
