# 第三章：設定與使用方式

---

## 1. 設定說明

### 1.1 application.yml

外部設定檔位於 `config/application.yml`：

```yaml
app:
  import-dir: ./import                         # 來源檔案目錄
  output-dir: ./output                         # 輸出目錄
  process-year: 115                            # 處理年度（民國年，Integer）
```

### 1.2 insurance-mapping.yml（險種歸屬對照設定）

外部設定檔位於 `config/insurance-mapping.yml`，定義所有險種代號、名稱及分類歸屬。修改此檔案即可新增或移除險種，**不需要重新編譯程式**。

```yaml
app:
  insurance:
    codes:                          # 險種代號清單 (順序 = Sheet1 明細表欄位順序)
      - code: "0100"
        short-name: "一年期住宅火險"
        full-name: "一年期住宅火災保險"
      # ... 共 33 筆

    categories:                     # 大類及子分類歸屬
      - name: "火險"
        number: "一"
        sub-categories:
          - name: "火險"
            codes: ["0100", "0200", "0300", "0400"]
      # ... 共 9 大類、16 子分類
```

#### 新增險種步驟

1. 在 `codes` 區塊新增一筆（注意順序 = Sheet1 明細表欄位順序）
2. 在 `categories` 中找到對應大類，將代號加入 `sub-categories` 的 `codes`
3. 重新執行 `run.bat` 即可

**範例 — 新增 3300 長照險（歸屬到健康險）：**

```yaml
# 1. codes 區塊新增：
- code: "3300"
  short-name: "長照險"
  full-name: "長期照顧保險"

# 2. categories 的健康險 sub-categories 改為：
- name: "健康險"
  codes: ["3000", "3100", "3300"]
```

#### 移除險種步驟

1. 從 `codes` 區塊移除該筆代號
2. 從 `categories` 對應子分類的 `codes` 陣列中移除該代號
3. 重新執行 `run.bat` 即可

> ⚠️ `insurance-mapping.yml` 透過 Spring Boot 的 `spring.config.import` 機制載入，由 `InsuranceMappingService` 解析並提供給各模組使用。

### 1.3 覆蓋優先順序

```
外部 config/application.yml > 環境變數 > 內建 application.yml
```

| 參數位置 | application.yml | 環境變數 |
|----------|-----------------|----------|
| 匯入目錄 | `app.import-dir` | `APP_IMPORT_DIR` |
| 輸出目錄 | `app.output-dir` | `APP_OUTPUT_DIR` |
| 處理年度 | `app.process-year` | `APP_PROCESS_YEAR` |

## 2. 使用方式

### 2.1 基本用法

1. 將所有來源 `.xlsx` 檔案放入 `import/{year}/Q{quarter}/` 目錄（例如 `import/115/Q1/`）
2. 設定 `config/application.yml` 中的 `process-year`（民國年）
3. 執行程式
4. 從 `output/{year}/Q{quarter}/` 取得產出報表（例如 `output/115/Q1/`）

### 2.2 多公司匯入

放入多個來源檔案，系統會自動合併：

```
import/115/Q1/
├── 01_115Q1_自留保費統計表.xlsx    # 台產
├── 05_115Q1_自留保費統計表.xlsx    # 富邦
├── 29_115Q1_自留保費統計表.xlsx    # 美國國際
└── 32_115Q1_自留保費統計表.xlsx    # 美商安達
```

輸出：`output/115/Q1/115年產險業務(Q1季自留)保費統計表.xlsx`（僅顯示這 4 家公司，其餘隱藏）

### 2.3 多季度匯入

將不同季度的檔案放入對應的季度目錄，系統會掃描 `import/{year}/Q{quarter}/` 下所有季度目錄：

```
import/115/
├── Q1/
│   ├── 29_115Q1_自留保費統計表.xlsx
│   └── 05_115Q1_自留保費統計表.xlsx
└── Q2/
    └── 29_115Q2_自留保費統計表.xlsx
```

輸出：每個季度各自產出，例如 `output/115/Q1/115年產險業務(Q1季自留)保費統計表.xlsx` 與 `output/115/Q2/115年產險業務(Q2季自留)保費統計表.xlsx`

### 2.4 去年同期對比

系統會自動從 `output/{year-1}/Q{quarter}/` 目錄讀取去年報表，無需額外設定。例如處理年度為 115、季度 Q1 時，系統會自動讀取：

```
output/114/Q1/114年產險業務(Q1季自留)保費統計表.xlsx
```

系統會自動讀取 Sheet2 的 T 欄值填入 U 欄（去年同期合計），並計算成長率。

> ⚠️ 若去年檔案不存在，U 欄留空，日誌記錄 WARN，不影響其他功能。
