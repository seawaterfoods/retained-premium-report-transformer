# 第三章：設定與使用方式

---

## 1. 設定說明

### 1.1 application.yml

外部設定檔位於 `config/application.yml`：

```yaml
app:
  import-dir: ./import                         # 來源檔案目錄
  output-dir: ./output                         # 輸出目錄
  process-year: 115                            # 處理年度（民國年，Integer）
```

### 1.2 覆蓋優先順序

```
外部 config/application.yml > 環境變數 > 內建 application.yml
```

| 參數位置 | application.yml | 環境變數 |
|----------|-----------------|----------|
| 匯入目錄 | `app.import-dir` | `APP_IMPORT_DIR` |
| 輸出目錄 | `app.output-dir` | `APP_OUTPUT_DIR` |
| 處理年度 | `app.process-year` | `APP_PROCESS_YEAR` |

---

## 2. 使用方式

### 2.1 基本用法

1. 將所有來源 `.xlsx` 檔案放入 `import/{year}/Q{quarter}/` 目錄（例如 `import/115/Q1/`）
2. 設定 `config/application.yml` 中的 `process-year`（民國年）
3. 執行程式
4. 從 `output/{year}Q{quarter}/` 取得產出報表（例如 `output/115Q1/`）

### 2.2 多公司匯入

放入多個來源檔案，系統會自動合併：

```
import/115/Q1/
├── 01_115(01-03)_自留保費統計表.xlsx    # 台產
├── 05_115(01-03)_自留保費統計表.xlsx    # 富邦
├── 29_115(01-03)_自留保費統計表.xlsx    # 美國國際
└── 32_115(01-03)_自留保費統計表.xlsx    # 美商安達
```

輸出：`output/115Q1/115年產險業務(Q1季自留)保費統計表.xlsx`（僅顯示這 4 家公司，其餘隱藏）

### 2.3 多季度匯入

將不同季度的檔案放入對應的季度目錄，系統會掃描 `import/{year}/Q{quarter}/` 下所有季度目錄：

```
import/115/
├── Q1/
│   ├── 29_115(01-03)_自留保費統計表.xlsx
│   └── 05_115(01-03)_自留保費統計表.xlsx
└── Q2/
    └── 29_115(01-06)_自留保費統計表.xlsx
```

輸出：`output/115Q2/115年產險業務(Q2季自留)保費統計表.xlsx`（檔名用最大季度）

### 2.4 去年同期對比

系統會自動從 `output/{year-1}Q{quarter}/` 目錄讀取去年報表，無需額外設定。例如處理年度為 115 時，系統會自動讀取：

```
output/114Q1/114年產險業務(Q1季自留)保費統計表.xlsx
```

系統會自動讀取 Sheet2 的 T 欄值填入 U 欄（去年同期合計），並計算成長率。

> ⚠️ 若去年檔案不存在，U 欄留空，日誌記錄 WARN，不影響其他功能。
