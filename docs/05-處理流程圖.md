# 第五章：處理流程圖

---

## 1. 主流程

```mermaid
flowchart TD
    START([開始]) --> SCAN["Step 1: 掃描輸入目錄<br/>找出所有 .xlsx 檔案"]
    SCAN --> PARSE["Step 2: 檔名解析<br/>提取公司別/年度/季度"]
    PARSE --> PRE_VAL{"Step 3: 前置驗證<br/>年度一致？重複公司？"}

    PRE_VAL -->|❌ 失敗| ABORT1([中止並記錄 ERROR])
    PRE_VAL -->|✅ 通過| READ["Step 4: 讀取來源 Excel<br/>提取各險種自留保費"]

    READ --> READ_OK{"讀取成功？"}
    READ_OK -->|❌ 任一失敗| ABORT2([中止並記錄 ERROR])
    READ_OK -->|✅ 全部成功| DATA_VAL["Step 5: 資料驗證<br/>數值/空值/異常檢查"]

    DATA_VAL --> DATA_OK{"驗證通過？"}
    DATA_OK -->|❌ 失敗| ABORT3([中止並記錄 ERROR])
    DATA_OK -->|✅ 通過| GROUP["Step 6: 按季度分組<br/>+ 險種歸類 (33→16)"]

    GROUP --> LASTYEAR["Step 7: 讀取去年報表<br/>（無檔案則跳過）"]
    LASTYEAR --> TMPL_CHK{"Step 8: 模板存在？"}

    TMPL_CHK -->|❌ 不存在| ABORT4([中止並記錄 ERROR])
    TMPL_CHK -->|✅ 存在| WRITE["Step 9-10: 寫入模板<br/>產出報表"]

    WRITE --> DONE([✅ 完成])

    style ABORT1 fill:#f44,color:#fff
    style ABORT2 fill:#f44,color:#fff
    style ABORT3 fill:#f44,color:#fff
    style ABORT4 fill:#f44,color:#fff
    style DONE fill:#4a4,color:#fff
```

## 2. 模板寫入細節流程

```mermaid
flowchart TD
    OPEN["開啟模板 .xlsx<br/>(保留公式+格式)"] --> MAP["建立公司代號→列號映射<br/>(讀取 A 欄)"]
    MAP --> S1["寫入 Sheet1<br/>各公司 × 33 險種"]
    S1 --> S2["寫入 Sheet2<br/>各公司 × 16 分類"]
    S2 --> U["寫入 U 欄<br/>(去年同期數值)"]
    U --> HIDE["隱藏無資料公司列<br/>(setZeroHeight=true)"]
    HIDE --> TITLE["更新動態標題<br/>(年度/季度/月份)"]
    TITLE --> RENAME["重新命名 Sheet"]
    RENAME --> RECALC["設定公式重新計算"]
    RECALC --> SAVE["儲存輸出 .xlsx"]

    subgraph preserve["🔒 不觸碰的公式區域"]
        F1["Sheet1 AK 欄: =SUM(D:AJ)"]
        F2["Sheet1 小計列: =SUBTOTAL(9,...)"]
        F3["Sheet2 T 欄: =SUM(D:S)"]
        F4["Sheet2 V 欄: =IF(U=0,'',(T/U)-1)"]
        F5["Sheet2 小計/總計列"]
    end

    style preserve fill:#e8f4fd,stroke:#2196F3
```
